# Кастомный прицел для Stalcraft X (Python / PyQt6)

Прозрачный оверлей прицела, который работает поверх игры Stalcraft X и не блокируется античитом.

## Функции
- 🎯 Загрузка собственного прицела в формате PNG/JPG.
- 📐 Изменение размера прицела прямо в игре.
- 🕹 Смещение по осям X и Y на лету с точностью до 1 пикселя.
- 💾 Автосохранение всех настроек при закрытии.

## Управление
- `Ctrl + Alt + M` — Открыть / Закрыть меню настроек (после закрытия фокус возвращается в игру).
- `Ctrl + Alt + Q` — Полный выход из программы.

## Инструкция по запуску
1. Установите Python.
2. Установите зависимости в терминале:
   ```bash
   pip install PyQt6 keyboard pywin32
   ```
3. Запустите игру Stalcraft в режиме **"окно без рамки(режим совместимости)"**
4. Запустите скрипт `crosshair.pyw` **от имени Администратора**.
